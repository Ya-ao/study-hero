# Study-Hero 微信小程序上线计划

## 一、项目概述

Study-Hero（学习英雄）是一款基于微信小程序的智能学习工具，包含前端小程序和后端服务两部分。上线需要完成微信小程序注册、后端服务部署、前端配置修改等步骤。

***

## 二、第一步：代码上传到 GitHub

### 2.1 初始化 Git 仓库

```bash
# 进入项目目录
cd "d:\WorkSpace\trae demo\study-hero"

# 初始化 Git 仓库
git init

# 添加所有文件（.gitignore 会排除敏感文件）
git add .

# 提交代码
git commit -m "Initial commit: Study-Hero 学习英雄小程序"
```

### 2.2 连接 GitHub 远程仓库

```bash
# 添加远程仓库
git remote add origin https://github.com/Ya-ao/study-hero.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 2.3 已创建的 .gitignore

已自动创建 `.gitignore` 文件，排除以下敏感文件：

* `.env` - 环境变量（包含 API 密钥等敏感信息）

* `node_modules/` - 依赖文件

* `project.private.config.json` - 私有配置

## 四、后端服务部署

### 4.1 服务器环境配置

```bash
# 安装 Node.js（推荐 v18 LTS）
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装 PM2 进程管理器
sudo npm install -g pm2

# 安装 Nginx（反向代理）
sudo apt-get install nginx
```

### 4.2 代码部署

1. **从 GitHub 克隆代码**

   ```bash
   git clone https://github.com/你的用户名/study-hero.git /var/www/study-hero
   cd /var/www/study-hero/server
   npm install --production
   ```

2. **配置环境变量**

   * 创建生产环境配置文件 `.env`

   * 填写真实的配置信息：

   ```
   PORT=3000
   NODE_ENV=production
   SUPABASE_URL=<your-supabase-url>
   SUPABASE_ANON_KEY=<your-supabase-key>
   WECHAT_APPID=<your-wechat-appid>
   WECHAT_SECRET=<your-wechat-secret>
   AI_API_KEY=<your-ai-api-key>
   AI_API_URL=https://ark.cn-beijing.volces.com/api/v3/chat/completions
   ```

### 4.3 PM2 启动服务

```bash
pm2 start app.js --name study-hero-server
pm2 save
pm2 startup
```

### 4.4 Nginx 反向代理配置

```nginx
server {
    listen 443 ssl http2;
    server_name your-domain.com;
    
    ssl_certificate /path/to/your/cert.pem;
    ssl_certificate_key /path/to/your/key.pem;
    
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

## 五、前端配置修改

### 5.1 修改 API 地址

修改 `miniprogram/config/index.js`：

```javascript
// 将 localhost 改为生产环境域名
const BASE_URL = 'https://your-domain.com'
```

### 5.2 配置合法域名

在微信公众平台配置：

* **request 合法域名**：`https://your-domain.com`

路径：登录微信公众平台 → 开发管理 → 开发设置 → 服务器域名

## 六、微信小程序发布流程

### 6.1 上传代码

1. 打开微信开发者工具
2. 点击右上角「上传」按钮
3. 填写版本号和项目备注

### 6.2 提交审核

1. 登录微信公众平台
2. 进入「管理」→「版本管理」
3. 点击「提交审核」

### 6.3 发布上线

审核通过后点击「发布」即可上线
