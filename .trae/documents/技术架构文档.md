```
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐              │

│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐       │
```

* **前端**：微信小程序，负责UI渲染和用户交互

* **后端**：独立Node.js服务，提供RESTful API接口

* **通信**：通过HTTPS协议进行数据交互

## 七、AI问答生成设计

### 7.1 Prompt模板

```
你是一位专业的教育专家，请围绕主题"{topic}"生成{count}道{difficulty}难度的选择题。

难度说明：
- easy: 基础知识
- medium: 中等难度
- hard: 较高难度

要求：
1. 每道题有4个选项，只有一个正确答案
2. 题目要有趣味性
3. 包含答案解析
4. 返回JSON格式

返回格式：
{
  "questions": [
    {
      "question": "问题内容",
      "options": ["选项A", "选项B", "选项C", "选项D"],

  ]
}
```

## 八、成就系统设计

### 8.1 成就规则

| 成就类型          | 名称     | 解锁条件      |
| ------------- | ------ | --------- |
| first\_answer | 初出茅庐   | 完成首次答题    |
| answer\_100   | 学海无涯   | 累计答题100道  |
| answer\_500   | 百题斩    | 累计答题500道  |
| answer\_1000  | 千题王    | 累计答题1000道 |
| streak\_7     | 连续7天   | 连续学习7天    |
| streak\_30    | 连续30天  | 连续学习30天   |
| perfect       | 满分达人   | 单次答题满分    |
| accuracy\_90  | 正确率90% | 单次正确率≥90% |

### 8.2 成就检查逻辑

```javascript
function checkAchievements(user, record) {
  const achievements = [];
  
  // 首次答题
  if (user.stats.totalQuestions === 0) {
    achievements.push({ type: 'first_answer', name: '初出茅庐', icon: '🌱' });
  }
  
  // 累计答题
  const totalAfter = user.stats.totalQuestions + record.totalQuestions;
  if (totalAfter >= 100 && user.stats.totalQuestions < 100) {
    achievements.push({ type: 'answer_100', name: '学海无涯', icon: '📚' });
  }
  
  // 满分
  if (record.score === 100) {
    achievements.push({ type: 'perfect', name: '满分达人', icon: '💯' });
  }
  
  // 正确率90%
  if (record.correctCount / record.totalQuestions >= 0.9) {
    achievements.push({ type: 'accuracy_90', name: '正确率90%', icon: '🎯' });
  }
  
  return achievements;
}
```
